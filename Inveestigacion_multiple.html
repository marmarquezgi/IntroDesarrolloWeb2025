Diferencias entre etiquetas en HTML

<b></b> vs. <strong></strong>

Visualmente:
Ambas muestran el texto en negritas. A simple vista no hay diferencia.

Semántica:
• <b> → Solo cambia el estilo (negritas), sin dar importancia al contenido.
• <strong> → Además de poner negritas, indica énfasis fuerte / importancia semántica.

Google: Tiende a dar más peso al texto dentro de <strong> que al de <b>, porque entiende que es relevante para el contexto de la página.

<i></i> vs. <em></em>

Visualmente:
Ambos muestran el texto en cursiva.

Semántica:
• <i> → Solo cambia el estilo (cursiva)
• <em> → Además de cursiva, da un énfasis semántico, como si "resaltara" una palabra clave en el discurso.

Google: El texto dentro de <em> puede interpretarse como más relevante que en <i>.

<s></s> vs. <del></del>

Visualmente:
Ambos muestran el texto con una línea tachada.

Semántica:
• <s> → Solo indica un cambio visual (tachado), normalmente para mostrar algo incorrecto o irrelevante.
• <del> → Además de tachar, indica que el texto fue eliminado o sustituido (ej. en actualizaciones de precios o versiones de un documento).

Google: Interpreta mejor <del> como contenido eliminado, lo que puede ser útil en sitios de e-commerce.

¿Qué son los Elementos de Bloque?

Un elemento de bloque es aquel que siempre comienza en una nueva línea y ocupa todo el ancho disponible de su contenedor (su elemento padre), estirándose hacia la izquierda y la derecha tanto como pueda.

5 ejemplos de Elementos de Bloque

  1. <div>: Se usa para agrupar contenido y crear layouts.
  2. <p> (Párrafo): Define un párrafo de texto. Siempre tiene margen por arriba y por abajo por defecto.
  3. <h1>, <h2>, ..., <h6> (Encabezados): Los títulos de sección. <h1> es el más importante y <h6> el menos importante.
  4. <ul>, <ol>, <li> (Listas):
    • <ul> para listas desordenadas (con viñetas)
    • <ol> para listas ordenadas (con números)
    • <li> para cada item de la lista
  5. <section>, <article>, <header>, <footer>, <nav> (Elementos semánticos de HTML5): Etiquetas que definen secciones específicas de una página, comportándose como bloques.

¿Qué son los Elementos de Línea?

Un elemento de línea (inline element) es aquel que no forza un salto de línea y solo ocupa el ancho que su contenido necesita.

5 ejemplos de Elementos de Línea

  1. <span>: El contenedor genérico en línea. Se usa para aplicar estilo a una parte específica de un texto o contenido.
  2. <a> (Anchor - Enlace): Define un hipervínculo.
  3. <strong> y <em>: Etiquetas semánticas para dar énfasis fuerte y énfasis, respectivamente.
  4. <img> (Image): Para insertar imágenes. Es un elemento de línea reemplazado y especial porque sí puedes cambiar su width y height.
  5. <br> (Line Break - Salto de línea): Forza un salto de línea dentro de un texto o un elemento de bloque. Es un elemento vacío (sin etiqueta de cierre).
